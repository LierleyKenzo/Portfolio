# Portfolio Professionnel - Kenzo Lierley
Bienvenue sur le dépôt de mon portfolio numérique. Ce projet a été réalisé dans le cadre de ma formation en BUT Informatique à l'IUT de [Nom de ton IUT]. L'objectif est de présenter mon parcours, mes réalisations techniques et de démontrer l'acquisition des compétences du référentiel national.

## Technologies utilisées
- Framework : Vue.js 3 (Composition API)
- Build Tool : Vite

## Structure du Projet
Le site est découpé en plusieurs sections stratégiques :

### Profil
Une brève introduction sur mes aspirations et ma personnalité.

### Stack Technique
Les langages, outils et frameworks que je maîtrise.

### Projets & Compétences
Présentation de mes travaux concrets, liés directement au référentiel de compétences de l'IUT.

### Parcours
Ma frise chronologique (Formation & Expériences).

### Contact
Formulaire et liens vers mes réseaux professionnels.

## Focus - Référentiel de Compétences (IUT)
Chaque projet présenté dans ce portfolio est tagué selon les compétences ciblées par le BUT :

### Réaliser
Développement d'applications.

Analyse algorithmique et efficacité.

### Administrer
Gestion des systèmes et réseaux.

### Gérer
Base de données et stockage.

Gestion de projet et méthodes agiles.

### Collaborer
Travail d'équipe et communication.

### Note
Chaque fiche projet détaille les livrables produits et le niveau d'autonomie atteint.

## Installation et Lancement
Si vous souhaitez cloner ce projet pour le tester en local :

### Cloner le dépôt
```bash
git clone https://github.com/ton-pseudo/portfolio-vue.git
```

### Installer les dépendances
```bash
npm install
```

### Lancer le serveur de développement
```bash
npm run dev
```

### Build pour la production
```bash
npm run build
```
